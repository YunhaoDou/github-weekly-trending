## About / Caveats

- Data source is `github.com/trending?since=weekly` — same weekly star count GitHub itself shows on its trending page.
- Coverage: overall page + 14 language pages, ~20 repos per page. Total uniques typically 150-250.
- **Project type** classification is rule-based (GitHub topics + name/description keywords). Edit `data/categories.yaml` to add or refine categories.
- Topic enrichment runs when `GITHUB_TOKEN` is available (always true in CI). Without it, classification falls back to name + description only.
- Star count is a popularity proxy, not a quality measure.
