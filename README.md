## About / Caveats

- **Weekly delta** = stars on `today` − stars on closest snapshot ~7 days earlier. We snapshot the top ~3000 repos by total stars daily.
- **Project type** classification is rule-based (keywords + GitHub topics). It is not perfect; expect occasional miscategorization.
- Repos that appear in this week's top-3000 but not 7 days ago are skipped (no baseline). They will surface in later weeks.
- Star count is a popularity proxy, not a quality measure.
- Snapshots older than 90 days are auto-pruned to keep the repo small.
